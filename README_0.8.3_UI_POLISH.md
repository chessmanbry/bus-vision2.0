# BusVision 2.0.0.8.3 UI Polish

本版針對採集畫面操作體驗做小改版：

- 採集畫面透明文字按鈕改成較精緻的可愛圖標按鈕。
- 上方狀態列的日夜、閃光、測距、更多設定改成圓角玻璃圖標。
- 底部重啟 / 錄影 / 拍照改成相機式圓形圖標操作。
- 變焦改成主畫面常駐滑桿，支援 1x 到 10x 快速調整，並保留 1x/2x/3x/5x/10x 快捷點選。
- 底部「採集 / 資料庫」導航加入 safe-area padding，避免被 Android 三鍵導航列遮住。

安裝前請先確認 `npm run build` 有成功產生 `dist/`，再執行 `npx cap sync android`。
